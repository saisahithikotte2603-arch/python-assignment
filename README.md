This repository contains two python programs.

## task 1
Perform basic mathematical operations
problem statement
1.Takes two numbers as input from user.
2.performs the  following operations:
   >Additiom
   >Subtraction
   >Multiplication
   >Division
3.Displays the result of each operation
Expected output(Example)
Enter first number: 10
Enter second number: 5
Addition: 15
Subtraction: 5
Multiplication: 50
Division: 2.0
## task2
Create a Personalized Greeting
Problem Statement
Write a Python program that
Takes the user's first name and last name as input.
Combines them into a full name.
Prints a personalized greeting using the full name.
Expected Output (Example)
Enter first name: Sai
Enter last name: Sahithi
Hello, Sai Sahithi! Welcome to Python Programming.
##How to Run the Programs
1.Make sure Python is installed on your system.
2.Download or clone this repository.
3.Open terminal / command prompt inside the project folder.
4.Run the programs:
python task1.py
python task2.py
